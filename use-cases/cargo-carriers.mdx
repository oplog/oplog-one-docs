# Cargo Carriers

## What are CargoCarriers?

**Definition**: Defines the cargo/shipping companies that OPLOG works with. Requests should be sent or read using these definitions.

CargoCarriers represent the logistics partners and shipping companies integrated with the OPLOG platform. These entities are essential for managing shipments, tracking deliveries, and coordinating logistics operations.

<Warning>
**Important Notes**

Since cargo companies can be updated, please make an API request for the current response to ensure you have the latest information.
</Warning>

## API Endpoint

Get the list of available cargo carriers:

```bash
GET /openapi/v1/CargoCarriers
```

## Common Use Cases

### 1. Populate Carrier Selection Dropdown

Create a user interface element for carrier selection:

```javascript
async function populateCarrierDropdown() {
  try {
    const response = await fetch('/openapi/v1/CargoCarriers');
    const data = await response.json();
    
    const selectElement = document.getElementById('carrier-select');
    selectElement.innerHTML = '<option value="">Select a carrier...</option>';
    
    data.items
      .filter(carrier => carrier.active) // Only show active carriers
      .forEach(carrier => {
        const option = document.createElement('option');
        option.value = carrier.key;
        option.textContent = carrier.value;
        option.title = carrier.description; // Tooltip with description
        selectElement.appendChild(option);
      });
  } catch (error) {
    console.error('Failed to load cargo carriers:', error);
  }
}
```

### 2. Validate Carrier Selection

Ensure the selected carrier is valid and active:

```javascript
async function validateCarrierSelection(carrierKey) {
  const carriers = await getCargoCarriers();
  const carrier = carriers.items.find(c => c.key === carrierKey);
  
  if (!carrier) {
    throw new Error('Invalid cargo carrier selected');
  }
  
  if (!carrier.active) {
    throw new Error('Selected cargo carrier is not currently active');
  }
  
  return carrier;
}
```

Show detailed carrier information to users:

```python Python
import requests
from typing import List, Dict, Optional

class CargoCarrierService:
    def __init__(self, base_url: str):
        self.base_url = base_url
        self._cache = None
        self._last_fetch = 0
        self.cache_duration = 3600  # 1 hour
    
    async def get_carriers(self) -> List[Dict]:
        """Get cargo carriers with caching"""
        import time
        
        current_time = time.time()
        if (self._cache is None or 
            (current_time - self._last_fetch) > self.cache_duration):
            
            response = requests.get(f"{self.base_url}/openapi/v1/CargoCarriers")
            response.raise_for_status()
            
            self._cache = response.json()['items']
            self._last_fetch = current_time
        
        return self._cache
    
    async def get_active_carriers(self) -> List[Dict]:
        """Get only active cargo carriers"""
        carriers = await self.get_carriers()
        return [c for c in carriers if c.get('active', False)]
    
    async def find_carrier_by_key(self, key: str) -> Optional[Dict]:
        """Find a specific carrier by key"""
        carriers = await self.get_carriers()
        return next((c for c in carriers if c['key'] == key), None)
```

## Error Handling

Always implement proper error handling for carrier-related operations:

```javascript
async function handleCarrierOperations() {
  try {
    const carriers = await getCargoCarriers();
    
    if (!carriers || !carriers.items || carriers.items.length === 0) {
      throw new Error('No cargo carriers available');
    }
    
    // Process carriers...
    
  } catch (error) {
    if (error.name === 'TypeError') {
      console.error('Network error - check API connectivity');
    } else if (error.status === 404) {
      console.error('CargoCarriers endpoint not found');
    } else {
      console.error('Failed to load cargo carriers:', error.message);
    }
    
    // Fallback behavior
    showErrorMessage('Unable to load cargo carriers. Please try again later.');
  }
}
```
